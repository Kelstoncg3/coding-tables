How To Define a Table

Tables are a structured set of data made up of rows and collumns.

  • Use <tr> to create a table row
  • Use <th> to create a table header (the text will be bold) in a table row
  • Use <td> to define where your table data is in a table row
  • Use <thead> to define where the head of your table is (for the browser)
  • Use <tfoot> to define where the foot of your table is (for the browser)
  • Use <caption> immediately after the <table> tag give a title to your table

Colspan vs. Rowspan

  • Colspan: makes a table cell span more than one collumn
  • Rowspan: makes a table cell span more than one row

Cell Spacing vs. Padding

  • Cell spacing is what defines the space between individual cells on your file.
  • Cell padding defines the space between a cell's border and the content in that cell. You can establish which side you want the space on, too, by specifying that the padding is on the left, right, top, or bottom.

How to Define Table Borders/Border Widths

  • In your CSS file, specify which tag you would like to style and then, inside of the {}, you can change the border-style, border-width, and border-color!
  • Outside of CSS, inside of the <table> tag, you can type border="#" to define how large you want the table borders.
